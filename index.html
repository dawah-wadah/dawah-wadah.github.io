Intro

Hi, I'm Wadah! I'm a NYC based full stack Web Developer with experience in Rails, React, Redux, and JavaScript. I've always known I wanted to code, so I started with CodeAcademy out of curiosity and the desire to learn a new skill. This quickly blossomed into a full on obsession. Sneaking in a few hours a day to code was not enough and so I landed at an intense coding bootcamp - AppAcademy. From there I was able to work on amazing projects that challenged every part of me. I believe every problem has a solution that can be found if we put in enough hard work, creative thinking, and coffee. After losing many hours of sleep I finally here to begin the next journey - one using my curiosity and creativity to create amazing apps. Feel free to check some of my awesome work out.

Skills

imGir

ImGir is a full-stack web application inspired by Imgur with an "Invader Zim" twist. It utilizes Ruby on Rails on the backend, a PostgreSQL database, and React.js with a Redux architectural framework on the frontend. Users are able to submit funny photos and have them viewed and commented on them by the community

Happy Bird

You thought you could escape this game, well think again. Happy Bird is a clone of the notorious Flappy Bird, built entirely on Javascript and Canvas

Domin8

DOMin8 is a lightweight JavaScript DOM Manipulation Library with tools to create, traverse, and manipulate DOM elements, set event handlers, and make front-end API requests. It reduces file size and loading speed by only implementing essential function.
